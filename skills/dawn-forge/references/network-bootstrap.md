# 网络引导范围

Dawn Engine V1 不负责安装或配置 Clash、代理订阅、TUN、离线 artifact 或第三方 installer。不要用已发布的内部模块拼装替代流程，也不要把订阅 URL、token、proxy URL 或私钥写入 Profile、Plan、Journal 或命令参数。

目标机无法访问 Catalog 软件所需的官方端点时，停止当前 Run，向用户报告网络前置条件。用户在 GUI 中独立完成网络配置并确认后，重新生成 Plan；不得复用网络条件变化前的批准。

随 Skill 保留的 artifact 与 private-input 脚本属于 V1 Engine 范围外的底层能力，不是当前公开装机工作流入口。
