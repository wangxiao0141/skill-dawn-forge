# 个人电脑配置

本上下文描述由一台现有电脑协助配置另一台个人电脑的装机过程，以及人和 Agent 在其中使用的核心概念。

## Language

**Dawn Forge**:
一个可独立安装的 Agent Skill，引导控制机上的 Agent 与用户协作完成目标机装机；它是全新实现，不是旧版装机程序的延续。
_Avoid_: 装机脚本、旧 Dawn Forge、controller

**控制机**:
当前运行 Agent、发起并协调装机过程的 macOS 或 Windows 个人电脑。
_Avoid_: 当前电脑、旧电脑、主电脑

**目标机**:
接受环境配置的 macOS 或 Windows 个人电脑；不包含企业 MDM 管理设备或无人值守服务器。
_Avoid_: 新电脑、远程电脑、被控机

**初始可用状态**:
目标机已完成操作系统安装和首次设置，用户能够以管理员身份登录并接入局域网；外网可以暂时不可用。
_Avoid_: 裸机、全新机器、已装机

**装机**:
目标机进入初始可用状态后，对远程访问、网络工具、软件和个人开发环境进行配置的过程；不包含 BIOS、磁盘分区或操作系统安装。
_Avoid_: 系统安装、刷机、初始化

**建联**:
控制机尚不能访问目标机时，用户只在目标机通过 GUI 开启 SSH 并读取局域网登录地址；Agent 生成控制机命令，用户在控制机输入目标机密码，最终建立经过验证的 SSH 公钥连接。
_Avoid_: 连一下、配 SSH、远程登录

**管理连接**:
装机完成后仍长期保留的控制机到目标机 SSH 公钥连接；控制机默认验证并复用 `~/.ssh/id_ed25519`，缺失时才创建。
_Avoid_: 临时 SSH、后门、永久会话

**首次信任**:
控制机首次通过用户从目标机系统界面读取的局域网地址连接时记录 SSH host key；之后只有 host key 保持一致才允许继续。
_Avoid_: 无条件信任、关闭 host-key 检查、自动清除 host key

**局域网目标**:
只能由同一局域网内的控制机通过稳定 SSH alias 或局域网域名访问的目标机，不通过公网暴露 SSH 服务。
_Avoid_: 公网主机、互联网服务器、固定 IP、端口转发目标

**平台路由**:
Agent 根据装机 profile 声明和 SSH 探测结果，在 macOS 与 Windows 的目标流程之间选择；声明与事实不一致时停止。
_Avoid_: Mac 模式、Windows 脚本、自动猜系统

**人工步骤**:
因需要物理操作、图形界面交互、密码输入或明确授权，只能由用户在目标机完成的装机步骤。
_Avoid_: 手动兜底、Agent 做不了的事

**代理客户端**:
装机 profile 按需声明的目标机网络代理应用；`Clash Verge Rev` 是可选实现，不是每次装机的固定前置项。
_Avoid_: 必装 Clash、默认代理、控制机代理

**装机 profile**:
描述目标平台、最终需要具备的软件、版本策略和设置的声明式配置；用户从 Agent 展示的候选中选择本次 profile，目标机 SSH alias 在运行时建立或复用，软件集合完全由所选 profile 决定。
_Avoid_: 装机清单、默认软件集、安装脚本、命令列表

**安装计划**:
受控 preflight 根据装机 profile、目标机现状、批量 metadata 和本次联网探测生成的完整动作集合；profile、目标身份和证据写入无秘密 receipt 并由 digest 绑定。用户一次确认计划与 digest，执行器再按依赖、联网位置、联网路由和安装机制拆分为多个安装批次。
_Avoid_: 单个软件确认、命令预览、安装脚本

**安装批次**:
安装计划确认后，由受控执行器生成并按顺序推进的执行单元；每批最多包含三个软件，批次内的执行模式、安装机制、联网位置、联网路由、权限模式和依赖层级必须一致。安装批次不是一条大命令，软件仍逐项安装和验证。
_Avoid_: 全量安装、一个 Homebrew 命令、下载队列

**联网路由**:
根据本次动作实际访问的 metadata 和 artifact 官方端点现场判定的 `direct`、`clash` 或 `local`；它属于运行时事实，不写入装机 profile，也不按软件名称或发布者所在地区猜测。
_Avoid_: 国内软件、国外软件、平台路由、固定代理分组

**联网位置**:
本次 endpoint probe 和下载实际发生的 `controller`、`target` 或 `none`；控制机和目标机的 `direct`/`clash` 状态分别记录和切换，不能互相代替。
_Avoid_: 当前网络、开着代理、统一 Clash 状态

**执行模式**:
安装动作由受控 runner 自动完成时为 `automated`；需要 GUI、管理员授权、重启或人工 installer 时为 `manual-receipt`。人工动作只有受控验收 receipt 通过后才解除依赖。
_Avoid_: 自动兜底、人工即成功、安装后再说

**运行状态**:
受控执行器在控制机原子保存的无秘密 journal，记录安装计划 digest、目标身份、批次、逐项 `fetch`、`install`、`verify` 结果和精确的活动进程所有权；它是恢复、状态观察和取消的事实来源。
_Avoid_: SSH 命令返回、进程名轮询、聊天记录状态

**运行入口**:
控制机按 `prepare → advance → observe → cancel` 管理安装 run 的深接口；manifest 一次绑定 profile、目标 identity receipt、preflight、schedule、SSH config、canonical cache 和 state 路径，后续不让 Agent 重填相关参数。
_Avoid_: 临时 SSH 命令、每批重新拼参数、多个后台 owner

**风险确认**:
批量安装之外，涉及秘密输入、图形界面授权、重启、删除或覆盖已有配置时，由用户单独作出的明确授权。
_Avoid_: 每步确认、默认同意、统一授权

**版本策略**:
普通软件默认使用最新稳定版，已满足要求时跳过；只有装机 profile 明确要求时才固定版本，不自动降级。
_Avoid_: 永远最新版、强制升级、统一锁定版本

**秘密输入**:
装机过程中由用户在控制机本地受保护终端或目标机 GUI 输入、不得进入聊天、命令参数、装机 profile、仓库、输出或运行状态的订阅地址、密码、token 和私钥。订阅地址等应用配置先保存到控制机用户目录下的受保护文件，再由 Agent 传到目标机并由用户手动应用；密码、token 和私钥仍由用户在目标机本地输入。
_Avoid_: 敏感配置、凭据字段、secret 配置

**配置交接**:
受控传输入口在控制机验证私有文件、Git/ACL 边界和目标 identity receipt，把用户提供的文字配置或现有配置文件以固定名称原子传到目标机固定目录，并核对 digest，供用户手动应用；Agent 不生成、猜测、自动导入或执行配置。
_Avoid_: 自动配置、剪贴板同步、配置中心

**外部服务密钥**:
按装机 profile 在目标机生成并由目标机持有的 SSH 用户密钥；不同用途的目标机身份彼此独立，也不与控制机访问目标机所用的管理身份复用。
_Avoid_: 目标机管理密钥、单一通用 SSH key、复制私钥
