# 个人电脑配置

本上下文描述由一台现有电脑协助配置另一台个人电脑的装机过程，以及人和 Agent 在其中使用的核心概念。

## Language

**Dawn Forge**:
一个可独立安装的 Agent Skill，引导控制机上的 Agent 与用户协作完成目标机装机；它是全新实现，不是旧版装机程序的延续。
_Avoid_: 装机脚本、旧 Dawn Forge、controller

**控制机**:
当前运行 Agent、发起并协调装机过程的电脑。
_Avoid_: 当前电脑、旧电脑、主电脑

**目标机**:
接受环境配置的新电脑；第一版仅指个人使用、搭载 Apple Silicon（M 系列）的 Mac mini，不包含 Intel Mac 或企业 MDM 管理设备。
_Avoid_: 新电脑、远程电脑、被控机

**初始可用状态**:
目标机已安装并完成 macOS 首次设置，用户能够以管理员身份登录，且目标机能够正常联网和使用。
_Avoid_: 裸机、全新机器、已装机

**装机**:
目标机进入初始可用状态后，对远程访问、网络工具、软件和个人开发环境进行配置的过程；不包含 BIOS、磁盘分区或操作系统安装。
_Avoid_: 系统安装、刷机、初始化

**建联**:
控制机尚不能访问目标机时，由 Agent 生成明确的本地操作步骤，用户在目标机执行必要命令和管理员授权，最终建立经过验证的 SSH 公钥连接。
_Avoid_: 连一下、配 SSH、远程登录

**管理连接**:
装机完成后仍长期保留的控制机到目标机 SSH 公钥连接，用于后续检查、修复、更新和重新执行装机清单。
_Avoid_: 临时 SSH、后门、永久会话

**首次信任**:
控制机首次通过目标机域名连接时自动记录 SSH host key，之后只有 host key 保持一致才允许继续连接。
_Avoid_: 跳过验证、人工验指纹、无条件信任

**局域网目标**:
只能由同一局域网内的控制机通过域名访问的目标机，不通过公网暴露 SSH 服务。
_Avoid_: 公网主机、互联网服务器、端口转发目标

**人工步骤**:
因需要物理操作、图形界面交互、密码输入或明确授权，只能由用户在目标机完成的装机步骤。
_Avoid_: 手动兜底、Agent 做不了的事

**代理客户端**:
目标机使用的网络代理应用；第一版固定为适用于 Apple Silicon macOS 的 `Clash Verge Rev`。
_Avoid_: Clash、Clash Verge、FlClash、代理软件

**装机清单**:
描述目标机最终需要具备的软件、版本策略和设置的声明式配置；它表达目标状态，不编排具体命令或执行顺序。
_Avoid_: 配置文件、安装脚本、命令列表

**安装计划**:
Agent 根据装机清单和目标机现状生成的待安装软件汇总；用户确认一次后，计划中的普通软件作为一个批次执行。
_Avoid_: 单个软件确认、命令预览、安装脚本

**风险确认**:
批量安装之外，涉及秘密输入、图形界面授权、重启、删除或覆盖已有配置时，由用户单独作出的明确授权。
_Avoid_: 每步确认、默认同意、统一授权

**版本策略**:
普通软件默认使用最新稳定版，已满足要求时跳过；只有装机清单明确要求时才固定版本，不自动降级。
_Avoid_: 永远最新版、强制升级、统一锁定版本

**秘密输入**:
装机过程中所需但不得写入装机清单、聊天记录、日志或仓库的订阅地址、密码、token 和私钥；只能由用户在目标机本地提供。
_Avoid_: 敏感配置、凭据字段、secret 配置

**外部服务密钥**:
在目标机生成并由目标机持有的 SSH 用户密钥，包括只用于 GitHub 的专用密钥和用于其他主机的通用密钥；二者彼此独立，也不与控制机访问目标机所用的管理密钥复用。
_Avoid_: 目标机管理密钥、单一通用 SSH key、复制私钥
